Неплохой сайт для проекта. Возможно, один из лучших...

Шаблон сайта
Сделан в программе Artisteer. Artisteer - классический генератор статичных сайтов. Аналоги - Adobe Muse, Template Toster. Все эти программы позволяют легко создавать красивые сайты в графической среде, имеют множество встроенных шаблонов.
Запущен на сервере

В качестве сервера используется nginx с минимальными настройками для статических веб-страниц. Чётенько.
Загружен в Docker



Проект контейнеризирован и запускается в Docker на Alpine 3.16. И это хорошо!

Другие варианты
